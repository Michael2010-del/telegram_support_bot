# 🤖 Телеграм-бот технической поддержки

Бот для автоматизации технической поддержки интернет-магазина "Продаем все на свете"

## 📌 О проекте

Этот телеграм-бот помогает автоматически отвечать на частые вопросы пользователей, направлять запросы в нужные отделы (программисты и отдел продаж) и хранить все обращения в базе данных.

**Что умеет бот:**
- Автоматически отвечать на частые вопросы (FAQ)
- Принимать запросы к специалистам (программисты и отдел продаж)
- Хранить все обращения в базе данных SQLite
- Уведомлять администратора о новых запросах
- Удобное меню с кнопками

## 🛠 Технологии

- Python 3.7+
- pyTelegramBotAPI - библиотека для Telegram
- SQLite - база данных

## 📦 Установка

**Шаг 1. Скачайте файлы проекта**

Вам нужны три файла:
- bot.py
- database.py
- faq_data.py

Положите их в одну папку.

**Шаг 2. Создайте виртуальное окружение**

Откройте терминал в папке с проектом и выполните:

```bash
python -m venv venv
```

**Шаг 3. Активируйте окружение**

Windows:
```bash
venv\Scripts\activate
```

Mac/Linux:
```bash
source venv/bin/activate
```

**Шаг 4. Установите библиотеку**

```bash
pip install pyTelegramBotAPI
```

## ⚙ Настройка

**1. Создайте бота в Telegram**

- Напишите @BotFather в Telegram
- Отправьте команду /newbot
- Придумайте имя бота (например: Поддержка Магазина)
- Придумайте username (должен заканчиваться на bot)
- Сохраните полученный токен (выглядит так: 1234567890:ABCdefGHIjklmNOPqrstUVwxyz)

**2. Узнайте свой Telegram ID**

- Напишите @userinfobot в Telegram
- Нажмите Start
- Скопируйте ваш ID (цифры)

**3. Вставьте данные в файл bot.py**

Откройте bot.py и найдите эти строки:

```python
BOT_TOKEN = "ТВОЙ_ТОКЕН_СЮДА"
admin_ids = [123456789]
```

Замените на ваши данные:

```python
BOT_TOKEN = "789456123:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw"
admin_ids = [987654321]
```

## 🚀 Запуск

В терминале, в папке с проектом, выполните:

```bash
python bot.py
```

Если всё работает, вы увидите:

```
🤖 Бот технической поддержки запускается...
✅ База данных инициализирована
✅ Бот успешно запущен!
```

Бот готов к работе! Напишите ему /start в Telegram.

## 📖 Как пользоваться

**Для пользователей:**

1. Напишите боту команду /start
2. Используйте кнопки меню:

   - ❓ Частые вопросы - получить ответы на популярные вопросы
   - 🔧 Программисты - отправить запрос, если не работает сайт или оплата
   - 📦 Отдел продаж - отправить запрос, если проблема с товаром
   - ℹ️ Информация о магазине - узнать о компании

**Для администраторов:**

1. Отправьте боту команду /admin
2. Используйте панель администратора:

   - 📊 Все запросы - посмотреть все обращения
   - 🛠️ Запросы к программистам - только технические проблемы
   - 📦 Запросы к продажам - только вопросы о товарах

## 🗄 База данных

База данных создаётся автоматически при первом запуске в файле `support_bot.db`

В ней хранится таблица `requests` со следующими полями:

| Поле | Описание |
|------|----------|
| id | Номер запроса |
| user_id | Telegram ID пользователя |
| user_name | Имя пользователя |
| department | Отдел (программисты/продажи) |
| question | Текст вопроса |
| status | Статус (новый/в работе/закрыт) |
| created_at | Дата и время создания |

## 📝 Как изменить FAQ

Откройте файл `faq_data.py` и отредактируйте словарь FAQ:

```python
FAQ = {
    "вопрос пользователя": "ответ бота",
    "как оформить заказ": "🛍️ Инструкция по оформлению...",
}
```

Сохраните файл и перезапустите бота.

## 🐛 Устранение проблем

**Бот не запускается, ошибка Timeout**

Проверьте интернет-соединение. Возможно, Telegram API заблокирован - используйте VPN.

**База данных не создаётся**

Убедитесь, что у вас есть права на запись в папке с проектом. База данных создаётся автоматически.

**Бот не отвечает на команды**

Проверьте, правильный ли токен вы вставили. Токен можно получить заново у @BotFather.

**Ошибка при установке библиотеки**

Обновите pip: `pip install --upgrade pip` и повторите установку.

## 📞 Поддержка

Если у вас возникли проблемы:
1. Проверьте правильность токена и ID администратора
2. Убедитесь, что все три файла находятся в одной папке
3. Перезапустите бота командой `python bot.py`
