# Support Bot Telegram

**Python • Telebot • SQLite • Техническая поддержка**

---

## О проекте

Телеграм-бот для автоматизации технической поддержки интернет-магазина "Продаем все на свете". Бот обрабатывает частые вопросы пользователей, направляет запросы в профильные отделы и сохраняет историю обращений в базе данных для дальнейшего анализа.

---

## Основные возможности

- Автоответы на FAQ - 8+ готовых ответов на популярные вопросы
- Отдел программистов - решение проблем с сайтом и оплатой
- Отдел продаж - консультации по товарам и возвратам
- База данных SQLite - хранение всех обращений пользователей
- Уведомления администратора - моментальное оповещение о новых запросах
- Панель администратора - просмотр и фильтрация обращений
- Интуитивное меню - удобные кнопки для всех действий

---

## Архитектура проекта

### Структура файлов:

```
├── bot.py              # Основной файл бота (обработчики, кнопки)
├── database.py         # Работа с SQLite (CRUD операции)
├── faq_data.py         # База вопросов и ответов
├── support_bot.db      # База данных (создаётся автоматически)
└── config.py           # Конфигурация (токен бота) - опционально
```

### Отделы поддержки:

1. Программисты - технические неисправности, ошибки на сайте, проблемы с оплатой
2. Отдел продаж - вопросы о товарах, возврат, обмен, повреждения

### Статусы запросов в БД:

- новый - только что поступивший запрос
- в работе - специалист взял в обработку
- закрыт - проблема решена

## Установка и запуск

### Требования:

- Python 3.7+
- Библиотека: pytelegrambotapi
- Токен бота от @BotFather
- Telegram ID администратора (узнать у @userinfobot)

### Шаги установки:

**1. Создайте проект:**

```bash
mkdir support_bot && cd support_bot
```

**2. Создайте виртуальное окружение:**

```bash
python -m venv venv
```

**Windows:**

```bash
venv\Scripts\activate
```

**Mac/Linux:**

```bash
source venv/bin/activate
```

**3. Установите зависимости:**

```bash
pip install pyTelegramBotAPI
```

**4. Создайте файлы проекта:**

- bot.py
- database.py
- faq_data.py

**5. Настройте конфигурацию в bot.py:**

```python
# В bot.py найдите и замените:
BOT_TOKEN = "ТВОЙ_ТОКЕН_СЮДА"
ADMIN_IDS = [123456789]

# Пример:
BOT_TOKEN = "789456123:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw"
ADMIN_IDS = [987654321]
```

**6. Запустите бота:**

```bash
python bot.py
```

## База данных

### Структура таблицы requests:

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER | Уникальный номер запроса |
| user_id | INTEGER | Telegram ID пользователя |
| user_name | TEXT | Имя пользователя |
| department | TEXT | Отдел (программисты/продажи) |
| question | TEXT | Текст обращения |
| status | TEXT | Статус (новый/в работе/закрыт) |
| created_at | TEXT | Дата и время создания |

### Основные функции database.py:

```python
init_db()                      # Создание таблиц
add_request()                  # Добавление запроса
get_all_requests()             # Получить все запросы
update_request_status()        # Изменить статус
get_requests_by_department()   # Фильтр по отделам
```

## База знаний бота (FAQ)

### Вопросы и ответы:

**КАК ОФОРМИТЬ ЗАКАЗ:**

Для оформления заказа, пожалуйста, выберите интересующий вас товар и нажмите кнопку "Добавить в корзину", затем перейдите в корзину и следуйте инструкциям для завершения покупки.

**СТАТУС ЗАКАЗА:**

Вы можете узнать статус вашего заказа, войдя в свой аккаунт на нашем сайте и перейдя в раздел "Мои заказы".

**ОТМЕНИТЬ ЗАКАЗ:**

Если вы хотите отменить заказ, пожалуйста, свяжитесь с нашей службой поддержки как можно скорее.

**ПОВРЕЖДЕННЫЙ ТОВАР:**

При получении поврежденного товара, пожалуйста, сразу свяжитесь с нашей службой поддержки и предоставьте фотографии повреждений.

**ТЕХНИЧЕСКАЯ ПОДДЕРЖКА:**

Вы можете связаться с нашей технической поддержкой через этого бота (нажмите "Программисты") или по телефону на сайте.

**ДОСТАВКА:**

Информацию о доставке вы можете найти на странице оформления заказа на нашем сайте.

**ВОЗВРАТ ТОВАРА:**

Вы можете вернуть товар в течение 14 дней с момента получения. Для этого свяжитесь с отделом продаж.

**КАК ОПЛАТИТЬ:**

Мы принимаем банковские карты (Visa, MasterCard), электронные кошельки и наличные при самовывозе.

## Особенности проекта

1. Простота использования - всё управление через кнопки, не нужно запоминать команды
2. Автоматизация - частые вопросы обрабатываются без участия человека
3. Структурированность - запросы распределяются по профильным отделам
4. Прозрачность - администратор видит все обращения и их статусы
5. Надёжность - SQLite гарантирует сохранность всех данных
6. Мгновенные уведомления - админ получает оповещение о каждом новом запросе

## Устранение проблем
